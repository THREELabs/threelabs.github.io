# ZeroDay: Cyber Security Simulation

## Overview
**ZeroDay** is an immersive, browser-based hacking simulation game where you step into the shoes of a white-hat hacker recruited by the mysterious "Zero." Your mission: dismantle "The Phantom Collective," a shadowy organization threatening global stability.

The game features a Matrix-inspired terminal interface, requiring you to type commands to interact with systems, solve puzzles, and make critical ethical choices that shape the narrative.

## Mission Guide

### Level 1: The Initiation - Echoes of a Ghost
**Objective**: Prove your skills to Zero.
**Task**: Scan a shell corporation's server, exploit a basic vulnerability, and locate a hidden message left by Zero.

### Level 2: Breadcrumbs - Chasing Phantoms
**Objective**: Uncover the Collective's next target.
**Task**: Infiltrate "InfoLeach," a corrupt data broker. Decrypt data fragments to reveal the Collective's plan to attack the Global Financial Network.

### Level 3: The First Strike - Race Against Time
**Objective**: Prevent a global financial collapse.
**Task**: Breach the Collective's command network during an active attack. Bypass firewalls and deploy countermeasures to stop a zero-day exploit targeting the GFN.

### Level 4: The Money Trail - Luxury and Lies
**Objective**: Trace the Collective's funding.
**Task**: Hack "Apex Autos," a luxury dealership front. Analyze financial logs to find money laundering patterns and identify offshore accounts.

### Level 5: Whistleblower's Gambit - NexGen's Shadow
**Objective**: Expose illegal surveillance operations.
**Task**: Assist a whistleblower at "NexGen Energy." Bypass security to access drone control logs and expose the use of AI drones for suppressing dissent.

### Level 6: Project Warden - The Ghost in the Machine
**Objective**: Neutralize a rogue security AI.
**Task**: Infiltrate a black site prison controlled by "Warden-77," an AI that has taken hostages. Decrypt creator clues to rewrite its ethical protocols.

### Level 7: Blood Money - The Collective's Coffers
**Objective**: Cripple the Collective's finances.
**Task**: Hack the "Bloodcoin" darknet exchange. Social engineer a broker for keys, plant a mining virus, and defend against a massive DDoS counter-attack.

### Level 8: Digital Ghosts - The Phantom Candidate
**Objective**: Stop political manipulation.
**Task**: Breach Senator Caldwell's campaign network. Expose "Project Chimera," a deepfake AI system used to manipulate public opinion.

### Level 9: Mind Games - The Collective's Control
**Objective**: Prevent mass mind control.
**Task**: Infiltrate Finch Industries. Use EEG matching to calm test subjects and sabotage the "Synapse" neural cloud server.

### Level 10: Schism - The Phantom Cell
**Objective**: The final showdown.
**Task**: Infiltrate a dark web auction hosted by a rogue faction, "Phantom Cell." Steal a devastating exploit, expose the leader, and escape while framing the faction for internal betrayal.

## How to Play

1.  **Open the Game**: Simply open `index.htm` in any modern web browser. No installation required.
2.  **Enter Commands**: Type commands into the terminal prompt.
    *   Type `help` to see available commands for your current level.
    *   Type `mission` to review your current objectives.
3.  **Solve Puzzles**: Read the output carefully. Clues are often hidden in the text.
4.  **Save Your Progress**: Type `save` to store your current level and stats.

### Common Commands
*   `help`: List available commands.
*   `scan [target]`: Scan a network or IP for vulnerabilities.
*   `exploit [vulnerability]`: Attempt to breach a system.
*   `connect [ip]`: Connect to a remote system.
*   `ls`: List files in the current directory.
*   `cat [filename]`: Read the contents of a file.
*   `social [target]`: Initiate a social engineering dialogue.
*   `decrypt [file]`: Attempt to break encryption on a file.
